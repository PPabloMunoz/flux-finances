# Security Policy

## Supported Versions

Currently, only the latest version of Flux Finances is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Flux Finances seriously. If you believe you have found
a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to [contact@pablomunoz.dev](mailto:contact@pablomunoz.dev)

Include as much information as possible:

- Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting)
- Location of the vulnerability
- Step-by-step instructions to reproduce the issue
- Potential impact

We will acknowledge your report within 48 hours and provide a timeline for a fix.
We ask that you follow responsible disclosure practices and give us time to
address the issue before making it public.
